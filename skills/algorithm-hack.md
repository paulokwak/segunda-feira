# Algorithm Hack

Engenharia reversa de algoritmos de distribuição para maximizar alcance orgânico.

## Framework por Plataforma

### YouTube
- **Shorts**: Cascata 200→2K→10K→100K, decisão em 24-72h
- **Longos**: CTR thumbnail >5%, retenção 30s >70%
- **Freq**: 4-5 Shorts + 1-2 longos/semana

### Instagram Reels
- **Métrica-chave**: Watch time proporcional (não absoluto)
- **Ouro**: Compartilhamentos via DM
- **Duração**: 7-15s (conclusão), 30-45s (densidade), 60-90s (narrativa)
- **Freq**: 5-7 Reels/semana

### TikTok
- **Cada vídeo começa do zero** (tamanho de conta irrelevante)
- **Peso**: watch time > rewatches > shares > saves > comments > likes
- **Duração ideal tech**: 30-90 segundos
- **Freq**: 1-2/dia

## Estrutura de Conteúdo
```
HOOK (0-3s) → CONTEXTO (3-15s) → DESENVOLVIMENTO → PAYOFF (últimos 3-5s)
```

## 7 Hooks que Funcionam
1. Dado chocante
2. Pergunta provocativa
3. Contradição
4. Resultado (prova social)
5. Curiosidade
6. Urgência
7. Empatia

## Output
- Plano semanal de conteúdo com hooks, formatos e CTAs
- Análise de métricas com diagnóstico
- Recomendações de otimização específicas por plataforma
