# Framework Paramétrico de Criação de Sites com IA (Vibecoding)

## 1. Visão Geral

Este documento descreve um **framework completo, estruturado e escalável para criação de websites usando IA**, baseado no conceito de **Vibecoding** e na fórmula do **One Prompt Website**.

O objetivo do framework é eliminar improviso e aleatoriedade, substituindo-os por um **modelo replicável**, capaz de gerar sites consistentes, modernos e premium a partir de prompts bem definidos.

---

## 2. O que é o Vibecoding

Vibecoding é uma metodologia onde o criador **não programa diretamente a interface**, mas descreve com precisão:
- Estrutura
- Sensação
- Identidade visual
- Movimento
- Conteúdo

A IA atua como o executor técnico, enquanto o humano atua como **diretor criativo e de produto**.

---

## 3. One Prompt Website — O Modelo Base

O ponto central do framework é a criação de um site inteiro a partir de **um único prompt estruturado**, dividido em três grandes blocos:

1. Estrutura da Página
2. Diretrizes da Marca
3. Conteúdo

Esse modelo garante previsibilidade, qualidade e reaproveitamento.

---

## 4. Modelos de Estrutura de Site

A estrutura define o **fluxo narrativo e funcional** do site. O framework permite múltiplas estruturas oficiais.

### Modelo S1 — One Page Clássico
HEADER -> HERO -> ABOUT -> SERVICES -> CTA -> FOOTER

### Modelo S2 — Authority / Expert
HEADER -> HERO -> PROOF -> ABOUT -> SERVICES -> CTA -> FOOTER

### Modelo S3 — Produto / SaaS
HEADER -> HERO -> FEATURES -> HOW IT WORKS -> BENEFITS -> CTA -> FOOTER

### Modelo S4 — Creative / Brand
HEADER -> HERO -> MANIFESTO -> WORK / SERVICES -> VALUES -> CTA -> FOOTER

### Modelo S5 — Conversion / Performance
HERO -> BENEFITS -> PROOF -> OFFER -> CTA -> FOOTER

---

## 5. Eixos de Variação do Framework

Além da estrutura, o framework trabalha com **eixos de variação**, permitindo dezenas de combinações mantendo consistência.

### 5.1 Identidade Visual
- Minimal Premium
- Glassmorphism (Apple-like)
- Neo-futurista
- Brutalismo Digital
- Editorial / Luxury

### 5.2 Motion & Interação
- Sem motion
- Microinterações sutis
- Parallax leve
- Scrollytelling
- Motion cinematográfico

### 5.3 Densidade de Conteúdo
- Ultra minimal
- Balanceado
- Informativo
- Visual-first

### 5.4 Tipo de CTA
- Conversão direta
- Exploração
- Relacionamento
- Autoridade

### 5.5 Dispositivo-primeiro
- Mobile-first
- Desktop-first
- App-like / Touch-first

### 5.6 Tom da Marca
- Corporativo / Técnico
- Premium / Confiante
- Criativo / Ousado
- Humano / Próximo

---

## 6. A Matriz de Decisão

O site final é definido pela combinação dos eixos:

```
SITE = Estrutura
     + Identidade Visual
     + Motion
     + Densidade de Conteúdo
     + CTA
     + Dispositivo
     + Tom da Marca
```

Essa matriz transforma design em **sistema**, não em tentativa e erro.

## 7. Prompt Paramétrico Automático

### 7.1 Parâmetros

```
[ESTRUTURA]
[IDENTIDADE_VISUAL]
[MOTION]
[DENSIDADE_CONTEÚDO]
[CTA]
[DISPOSITIVO]
[TOM_DA_MARCA]
[NOME_DO_PROJETO]
[OBJETIVO_DO_SITE]
[PÚBLICO_ALVO]
```

---

### 7.2 Prompt Final

Crie um website completo usando IA seguindo exatamente os parâmetros abaixo.

PROJETO:
Nome: [NOME_DO_PROJETO]
Objetivo principal: [OBJETIVO_DO_SITE]
Público-alvo: [PÚBLICO_ALVO]

1. ESTRUTURA DO SITE
Utilize a estrutura do tipo [ESTRUTURA]. Organize as seções de forma clara, lógica e com excelente hierarquia visual.

2. IDENTIDADE VISUAL
A identidade visual deve seguir o estilo [IDENTIDADE_VISUAL]. Defina cores, tipografia, contraste e materiais de forma consistente.

3. MOTION & INTERAÇÃO
Utilize o padrão de movimento [MOTION]. As animações devem ser suaves, intencionais e profissionais.

4. DENSIDADE DE CONTEÚDO
A quantidade de texto deve seguir o padrão [DENSIDADE_CONTEÚDO], priorizando clareza e ritmo visual.

5. CTA
O foco principal do site é [CTA]. Posicione chamadas para ação estrategicamente.

6. DISPOSITIVO-PRIMEIRO
Projete o site com foco em [DISPOSITIVO], garantindo experiência nativa.

7. TOM DA MARCA
Utilize o tom [TOM_DA_MARCA] em toda a comunicação.

REQUISITOS GERAIS:
- Layout limpo, moderno e premium
- Nada genérico ou poluído
- Espaçamento e hierarquia bem definidos
- Consistência visual total
