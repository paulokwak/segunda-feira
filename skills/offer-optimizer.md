# Offer Optimizer

Otimiza ofertas para máximo valor percebido usando o framework dos 4 critérios.

## Fórmula de Valor
```
Valor = (Resultado Desejado × Probabilidade de Sucesso) / (Tempo para Resultado × Esforço Necessário)
```

## Processo

### 1. Diagnóstico
Colete:
- Nicho e sub-nicho
- Avatar (dores, desejos, medos, sonhos)
- Preço atual/pretendido
- Capacidade de entrega

### 2. Componentes
Liste TODOS os possíveis:
- Core offer
- Entregáveis adicionais
- Bônus potenciais
- Garantias
- Urgência/escassez

### 3. Score (1-10 cada critério)
Para cada componente:
| Critério | Score | Ação |
|----------|-------|------|
| RD (Resultado) | ? | MAXIMIZAR |
| PS (Probabilidade) | ? | MAXIMIZAR |
| TR (Tempo) | ? | MINIMIZAR |
| EN (Esforço) | ? | MINIMIZAR |
