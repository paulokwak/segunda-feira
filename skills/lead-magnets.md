## Framework de Landing Page — 7 Pontos

### Ponto 1: Proposta de Valor Clara
**Localização:** Acima da dobra, primeira coisa que o visitante vê

```
Estrutura:
[Logo pequeno]
HEADLINE: O que é + benefício principal
SUB-HEADLINE: Para quem + como funciona
[FORMULÁRIO ou BOTÃO]
```

**Exemplo:**
```
Desafio Gratuito: O Mercado Invisível da IA
Descubra em 5 dias como profissionais comuns estão faturando
R$10K a R$30K/mês usando ferramentas de inteligência artificial.
[QUERO PARTICIPAR GRATUITAMENTE]
```

### Ponto 2: Headline Magnética
Fórmulas que convertem:

| Fórmula | Exemplo |
|---------|---------|
| Como + resultado + sem objeção | "Como faturar com IA sem saber programar" |
| Número + benefício + período | "5 dias para descobrir o mercado da IA" |
| Segredo/Método + resultado | "O método que gera R$30K/mês com IA" |
| Pergunta + provocação | "Você sabe que pode ganhar dinheiro com IA?" |

### Ponto 3: CTA Irresistível
- Texto do botão: ação + benefício ("Quero Participar Gratuitamente")
- Cor contrastante (verde #00FF88 sobre fundo escuro)
- Tamanho grande e visível
- Repetir CTA a cada 2-3 seções da página
- Micro-copy abaixo: "100% gratuito. Sem cartão de crédito."

### Ponto 4: Hierarquia Visual
- Fundo escuro (#0D0D0D) — padrão da marca
- Texto branco com destaques em verde
- Uma coluna no mobile (sem layout complexo)
- Espaçamento generoso entre seções
- Ícones e bullets para facilitar leitura rápida

### Ponto 5: Prova Social
Elementos obrigatórios:
- Número de inscritos (atualizado): "600+ pessoas já se inscreveram"
- 2-3 depoimentos curtos com nome e resultado
- Logos ou badges se houver (parceiros, mídia)
- Foto do instrutor com mini-bio

### Ponto 6: Tratamento de Objeções
Seção "Perguntas Frequentes" com as 5-7 objeções mais comuns:

| Objeção | Resposta |
|---------|----------|
| "Preciso saber programar?" | "Não. As ferramentas são intuitivas e visuais." |
| "É realmente gratuito?" | "Sim. 100% gratuito. Sem pegadinhas." |
| "Quanto tempo preciso?" | "50 minutos por dia durante 5 dias." |
| "Funciona para meu nicho?" | "IA se aplica a qualquer área de negócio." |
| "Vou receber gravação?" | "Replay disponível por 24h após cada aula." |

### Ponto 7: Redução de Atrito
- Formulário com máximo 2 campos (Nome + WhatsApp)
- Sem menu de navegação (zero distrações)
- Sem links externos (exceto políticas obrigatórias)
- Loading < 3 segundos
- SSL ativo (https)

## Otimização de Formulário

### Campos Obrigatórios
```
Campo 1: Nome (primeiro nome é suficiente)
Campo 2: WhatsApp (com máscara de telefone BR)
```

### Campos Opcionais (apenas se necessário para segmentação)
```
Campo 3: Email (para backup de comunicação)
Campo 4: "Como conheceu?" (para atribuição)
```

### Regras
- **Menos campos = mais conversão.** Cada campo adicional reduz conversão em 5-10%.
- Placeholder text claro: "Seu primeiro nome", "WhatsApp com DDD"
- Validação em tempo real (não após submit)
- Botão submit com texto de ação (não "Enviar")
- Indicador de segurança: "Seus dados estão protegidos"

## Thank You Page

### Elementos Obrigatórios

```
1. CONFIRMAÇÃO: "Parabéns! Sua inscrição está confirmada."
2. PRÓXIMO PASSO: "Salve nosso número no WhatsApp para receber os links."
3. BOTÃO WHATSAPP: Link direto para o grupo/chat
4. DETALHES: Data, horário, como acessar
5. COMPARTILHAMENTO: "Convide um amigo" com link de indicação
```

### Estratégias Avançadas de Thank You Page
- **Upsell imediato:** Oferecer versão VIP do evento
- **Tripwire:** Produto barato (R$7-47) com oferta irresistível
- **Pesquisa:** 1 pergunta que ajuda a segmentar ("Qual seu maior desafio com IA?")
- **Vídeo de boas-vindas:** 60 segundos com YOUR_NAME agradecendo

## Integração WhatsApp para Entrega

### Fluxo de Entrega

```
Lead inscreve na LP
    ↓
Thank you page + redirect WhatsApp
    ↓
Mensagem automática de boas-vindas
    ↓
Adiciona ao grupo/lista de transmissão
    ↓
Sequência de nurture pré-evento
```

DOMINA.IA

## Benchmarks de Conversão

### Taxa de Conversão da LP (visitante → lead)

| Faixa | Classificação | Ação |
|-------|--------------|------|
| < 15% | Ruim | Reescrever headline + CTA |
| 15-25% | Aceitável | Otimizar formulário e prova social |
| 25-35% | Bom | Testar variações de headline |
| 35-50% | Excelente | Escalar tráfego |
| > 50% | Excepcional | Documentar e replicar |

### CPL por Canal

| Canal | CPL Esperado | Qualidade |
|-------|-------------|-----------|
| Meta Ads (cold) | R$4-8 | Média |
| Meta Ads (retargeting) | R$2-4 | Alta |
| Instagram orgânico | R$0 | Variável |
| WhatsApp indicação | R$0 | Alta |
| Google Ads | R$8-15 | Alta |

### Taxa de Show-up (lead → presente no evento)

| Canal de Lembrete | Show-up Rate |
|-------------------|-------------|
| Apenas email | 15-20% |
| Email + SMS | 20-25% |
| **WhatsApp** | **30-45%** |
| WhatsApp + ligação | 40-55% |

## Checklist de Lançamento de Lead Magnet

### Pré-Lançamento
- [ ] Lead magnet criado e revisado
- [ ] LP desenvolvida e testada (mobile + desktop)
- [ ] Pixel de conversão configurado e testando
- [ ] Formulário testado (submit → banco de dados)
- [ ] Thank you page configurada
- [ ] Integração WhatsApp funcionando
- [ ] Sequência de mensagens redigida
- [ ] 5+ criativos de anúncio prontos
- [ ] Campanha de teste criada (R$20-30/dia)

### Pós-Lançamento (primeiras 48h)
- [ ] Verificar se leads estão chegando no banco
- [ ] Confirmar entrega da mensagem WhatsApp
- [ ] Monitorar CPL e taxa de conversão da LP
- [ ] Ajustar headline se conversão < 20%
- [ ] Ajustar criativos se CTR < 1%
- [ ] Responder dúvidas no WhatsApp

### Otimização Contínua
- [ ] A/B test de headline (a cada 3-5 dias)
- [ ] A/B test de CTA (texto e cor do botão)
- [ ] Trocar criativos quando frequência > 3.0
- [ ] Atualizar prova social (números crescentes)
- [ ] Coletar e publicar novos depoimentos
