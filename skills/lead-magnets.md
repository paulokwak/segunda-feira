## Framework de Landing Page — 7 Pontos

### Ponto 1: Proposta de Valor Clara
**Localizacao:** Acima da dobra, primeira coisa que o visitante ve

```
Estrutura:
[Logo pequeno]
HEADLINE: O que e + beneficio principal
SUB-HEADLINE: Para quem + como funciona
[FORMULARIO ou BOTAO]
```

### Ponto 2: Headline Magnetica

| Formula | Exemplo |
|---------|---------|
| Como + resultado + sem objecao | "Como faturar com IA sem saber programar" |
| Numero + beneficio + periodo | "5 dias para descobrir o mercado da IA" |
| Segredo/Metodo + resultado | "O metodo que gera R$30K/mes com IA" |
| Pergunta + provocacao | "Voce sabe que pode ganhar dinheiro com IA?" |

### Ponto 3: CTA Irresistivel
- Texto do botao: acao + beneficio ("Quero Participar Gratuitamente")
- Cor contrastante (verde #00FF88 sobre fundo escuro)
- Tamanho grande e visivel
- Repetir CTA a cada 2-3 secoes da pagina
- Micro-copy abaixo: "100% gratuito. Sem cartao de credito."

### Ponto 4: Hierarquia Visual
- Fundo escuro (#0D0D0D) — padrao da marca
- Texto branco com destaques em verde
- Uma coluna no mobile (sem layout complexo)
- Espacamento generoso entre secoes
- Icones e bullets para facilitar leitura rapida

### Ponto 5: Prova Social
- Numero de inscritos (atualizado)
- 2-3 depoimentos curtos com nome e resultado
- Logos ou badges se houver (parceiros, midia)
- Foto do instrutor com mini-bio

### Ponto 6: Tratamento de Objecoes
Secao "Perguntas Frequentes" com as 5-7 objecoes mais comuns

### Ponto 7: Reducao de Atrito
- Formulario com maximo 2 campos (Nome + WhatsApp)
- Sem menu de navegacao (zero distracoes)
- Sem links externos (exceto politicas obrigatorias)
- Loading < 3 segundos
- SSL ativo (https)

## Otimizacao de Formulario

### Campos Obrigatorios
```
Campo 1: Nome (primeiro nome e suficiente)
Campo 2: WhatsApp (com mascara de telefone BR)
```

### Regras
- **Menos campos = mais conversao.** Cada campo adicional reduz conversao em 5-10%.
- Placeholder text claro: "Seu primeiro nome", "WhatsApp com DDD"
- Validacao em tempo real (nao apos submit)
- Botao submit com texto de acao (nao "Enviar")
- Indicador de seguranca: "Seus dados estao protegidos"

## Thank You Page

### Elementos Obrigatorios

```
1. CONFIRMACAO: "Parabens! Sua inscricao esta confirmada."
2. PROXIMO PASSO: "Salve nosso numero no WhatsApp para receber os links."
3. BOTAO WHATSAPP: Link direto para o grupo/chat
4. DETALHES: Data, horario, como acessar
5. COMPARTILHAMENTO: "Convide um amigo" com link de indicacao
```

## Benchmarks de Conversao

### Taxa de Conversao da LP (visitante → lead)

| Faixa | Classificacao | Acao |
|-------|--------------|------|
| < 15% | Ruim | Reescrever headline + CTA |
| 15-25% | Aceitavel | Otimizar formulario e prova social |
| 25-35% | Bom | Testar variacoes de headline |
| 35-50% | Excelente | Escalar trafego |
| > 50% | Excepcional | Documentar e replicar |
